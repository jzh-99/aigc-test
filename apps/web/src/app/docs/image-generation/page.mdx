# 图片生成

图片生成位于**创作生成**页面（侧边栏 → 创作生成），默认进入图片模式。支持纯文字描述生成图片，也可上传参考图进行图生图创作。

![图片生成页面](/docs-images/image-gen-overview.png)

---

## 页面布局

页面分为左右两栏：

- **左侧**：生成面板，包含参考图上传、提示词输入、模型/参数配置和生成按钮
- **右侧**：历史记录，实时显示当前工作区的所有生成批次

---

## 参考图（图生图）

生成面板顶部为参考图上传区域，支持：

- 点击上传，或直接将图片**拖拽**到面板上
- 最多上传 **10 张**参考图，单张不超过 **20 MB**
- 已上传参考图后，点击图片堆叠预览可打开管理对话框，支持逐张删除或一键清空

上传参考图后，AI 将在生成时参考这些图片的风格、构图或内容。不上传参考图则为纯文生图模式。

## 提示词输入

在文本框中描述您想要生成的画面内容。

- 支持中英文
- 快捷键：**Ctrl+Enter**（Mac 为 Cmd+Enter）直接提交生成
- 可借助 [AI 助手](/docs/ai-assistant) 辅助撰写或优化提示词

---

## 可用模型与积分消耗

{/* TODO: 截图 - 模型选择下拉展开状态，展示两个模型选项 */}
![模型选择](/docs-images/image-gen-models.png)

| 模型 | 特点 | 积分消耗 |
|---|---|---|
| **全能图片2** | 快速生成，适合日常使用 | 5 积分/张 |
| **全能图片Pro** | 高质量输出，细节丰富 | 10 积分/张 |

---

## 参数配置

{/* TODO: 截图 - 参数配置区域，展示质量、比例选择器 */}
![参数配置](/docs-images/image-gen-params.png)

| 参数 | 选项 | 说明 |
|---|---|---|
| **质量** | 1K / 2K / 4K | 输出图片的分辨率等级，越高越清晰，积分消耗相同 |
| **比例** | 1:1 / 4:3 / 3:4 / 16:9 / 9:16 | 输出图片的宽高比 |
| **数量** | 1 / 2 / 3 / 4 张 | 单次批量生成的图片数量 |

---

## 批量生成

将**数量**设置为 2–4 张，单次提交即可批量生成多个结果，便于对比选优。

> **注意**：批量生成按数量倍数消耗积分。例如选择全能图片Pro + 4张 = 40 积分。提交前面板右下角会显示本次预计消耗积分，请确认余额充足。

---

## 提交与查看结果

填写提示词后，点击右下角**生成**按钮（或按 Ctrl+Enter）提交任务。

提交后，右侧**历史记录**面板顶部会出现"生成中"标记，实时显示任务进度。

{/* TODO: 截图 - 右侧历史记录面板，展示生成中状态的批次卡片 */}
![历史记录](/docs-images/image-gen-history.png)

> **提示**：任务不会因页面操作中断。提交后切换页面、切换功能模块，已提交的任务仍在后台继续运行，完成后结果保存在历史记录中。

点击历史记录中的任意批次，弹出详情面板，可查看该批次的所有生成图片，并对每张图片进行：

{/* TODO: 截图 - 批次详情面板，展示图片网格和操作按钮 */}
![批次详情](/docs-images/image-gen-detail.png)

| 操作 | 说明 |
|---|---|
| **放大预览** | 全屏查看高清原图 |
| **下载** | 将图片保存至本地 |
| **复用** | 将该批次的提示词和所有参数填回生成面板，在此基础上继续调整 |

**复用**功能是提升创作效率的核心工具——找到满意的风格后，复用参数进行微调，无需从头配置。
