# AI 助手

AI 助手是平台的**常驻悬浮按钮**，位于页面右下角。点击按钮即可打开 AI 助手面板，面板可自由拖拽移动，不影响其他页面操作。

![AI 助手悬浮按钮](/docs-images/ai-assistant-button.png)

## 两个功能标签

AI 助手面板顶部提供两个切换标签：

- **对话助手**：多轮自然语言对话，协助创意构思和提示词撰写
- **图片解析**：上传图片，AI 自动解析为结构化提示词

---

## 对话助手

在对话框中输入您的需求，助手以多轮对话方式给出专业建议或直接生成可用的提示词内容。

{/* TODO: 截图 - AI 助手对话界面，展示一段对话示例 */}
![对话助手](/docs-images/ai-assistant-chat.png)

**典型使用场景：**

- **提示词撰写与优化**：描述您的创意方向，助手帮您生成专业的图片/视频提示词
- **创意策划**：规划宣传短片分镜、海报设计方案、视觉风格定义
- **提示词改写**：将已有提示词按照特定风格或要求进行调整
- **内容翻译**：将中文创意描述转化为英文提示词，提升生成质量

> 在对话框中按 **Shift+Enter** 可换行，按 **Enter** 直接发送。助手回复内容右上角提供**复制**按钮，方便将提示词一键复制到生成面板。

---

## 图片解析功能

切换至**图片解析**标签，上传任意图片，点击**开始解析**按钮，助手将对图片进行深度视觉分析，输出一份结构化的 JSON 规格文档，可直接用于指导 AI 重新生成风格相似但内容不同的图片。

### 解析结果结构

解析完成后，助手将输出包含以下模块的 JSON 文档：

{/* TODO: 截图 - 图片解析结果，展示完整的 JSON 输出 */}
![图片解析结果](/docs-images/ai-assistant-parse-result.png)

| JSON 模块 | 内容说明 |
|---|---|
| `metadata` | 图片主题、尺寸、渲染引擎、色彩模式等基础元数据 |
| `typography_layer_specification` | 文字内容、字体样式、视觉效果、位置坐标等排版规格 |
| `pixel_level_visual_specs` | 角色/主体细节、背景场景、光照设置等像素级视觉描述 |
| `color_palette_hex` | 图片的主要色彩方案（十六进制色值） |
| `ai_generation_prompts` | 多格式 AI 生成提示词（见下表） |

**`ai_generation_prompts` 包含的提示词格式：**

| 字段 | 说明 |
|---|---|
| `dalle_3_optimized` | 针对 DALL·E 3 优化的英文提示词 |
| `midjourney_v6` | 针对 Midjourney v6 格式的提示词 |
| `chinese_description` | 中文创意描述，便于理解和二次修改 |

### 操作步骤

**第一步**：点击页面右下角的 AI 助手悬浮按钮，打开面板，切换至**图片解析**标签。

**第二步**：在上传区域选择或拖入您希望解析的参考图片，点击**开始解析**。

{/* TODO: 截图 - 图片解析标签，已上传图片准备解析的状态 */}
![上传图片](/docs-images/ai-assistant-upload.png)

**第三步**：获取 JSON 结果后，您可以：

- 直接复制 `ai_generation_prompts` 中的提示词用于图片生成
- 切换到**对话助手**标签，通过对话进一步修改 JSON 中的特定字段（如更换配色、调整文字内容）
- 将修改后的提示词用于批量生成，探索不同的设计变体

> 图片解析功能特别适合**借鉴参考图设计风格**的场景——解析后的 JSON 提示词能够还原设计语言，但生成的新图片在内容上与原图完全不同，规避版权风险。
