# 使用样例：海报制作

本章通过一个完整的实战案例，演示如何综合运用平台的**图片生成**、**AI 助手**功能，从参考图出发，制作一张风格相似但内容全新的定制化海报。

![海报制作案例展示](/docs-images/case-poster-result.jpg)

---

## 工作流程概览

```
参考海报
   ↓
AI 助手：图片解析 → JSON 提示词
   ↓
对话改写 JSON（方便在原提示词基础上改写）
   ↓
图片生成：粘贴提示词 + 上传参考图 + 批量生成
   ↓
选优下载
```

---

## 第一步：准备参考海报

找到一张设计风格符合您需求的参考海报。

{/* TODO: 截图 - 参考海报示例图 */}
![参考海报](/docs-images/case-poster-reference.jpg)

---

## 第二步：AI 助手解析参考图

点击页面右下角的 **AI 助手**悬浮按钮，切换至**图片解析**标签，上传参考海报，点击**开始解析**：

{/* TODO: 截图 - AI 助手图片解析标签，已上传参考图准备解析 */}
![AI 解析海报](/docs-images/case-poster-parse.png)

助手将返回包含配色、排版、视觉风格和多格式提示词的 JSON 文档。

---

## 第三步：通过对话改写 JSON

切换至**对话助手**标签，告诉助手您需要修改的内容，例如：

> 请将 JSON 中的品牌名称替换为"Toby.AI"，主色调改为深蓝色系（#0A1628），文案改为："用 AI，重新定义创作"，保持整体排版风格不变。

{/* TODO: 截图 - 对话改写 JSON 的过程，展示修改指令和助手返回的新 JSON */}
![改写 JSON](/docs-images/case-poster-rewrite.png)

您可以多轮对话反复调整，直到 JSON 中的各项参数符合您的预期为止。

---

## 第四步：生成海报

复制改写后 JSON 中 `ai_generation_prompts` 下的提示词，进入**创作生成**页面（图片模式）：

1. 选择合适的图像生成模型
2. 将提示词粘贴到输入框
3. 如有必要，在参考图区域上传参考图以保留构图风格
4. 设置目标尺寸（比例选择与参考图一致）
5. 将**数量**设为 4，开启批量生成

{/* TODO: 截图 - 图片生成页面，填好提示词和参数，即将提交的状态 */}
![图片生成配置](/docs-images/case-poster-generate.png)

提交后，在历史记录中查看 4 张生成结果，选取最满意的一张下载。

## 小结

通过以上流程，您实现了：借鉴参考图的设计语言和视觉风格，替换为自己的品牌信息和创意内容，生成的最终图片在版权上与参考图完全独立。
