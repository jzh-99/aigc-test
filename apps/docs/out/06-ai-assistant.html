6. AI 助手

AI 助手

AI 助手是平台内置的智能创作辅助工具，以悬浮按钮形式常驻于页面右下角，随时可用，无需跳转页面。

打开与关闭

点击右下角的机器人图标按钮，AI 助手面板从右侧弹出。再次点击或点击面板外区域可关闭面板。

悬浮按钮支持拖拽移动，可将其拖到屏幕任意位置。面板本身也支持拖拽移动和左右拖拽调整宽度（360–800px）。

两个功能标签

面板底部提供两个标签：

对话助手

自由对话模式，支持多轮对话，可协助您完成以下创作任务：

  • 提示词撰写与优化：描述您的创意方向，助手帮您生成专业的图片/视频提示词
  • 创意策划：规划宣传短片分镜、海报设计方案、视觉风格定义
  • 提示词改写：将已有提示词按照特定风格或要求进行调整
  • 内容翻译：将中文创意描述转化为英文提示词，提升生成质量

附加图片： 对话框右侧提供图片上传按钮，可在发送消息时附带一张图片，让助手结合图片内容给出建议。

发送快捷键： Shift+Enter 发送消息。

复制回复： 鼠标悬停在助手回复上，右下角出现复制按钮，点击即可复制全文。

清空对话： 点击面板顶部右侧的垃圾桶图标清空当前对话记录。

图片解析

上传任意图片，助手将对其进行深度视觉分析，输出一份结构化的 JSON 规格文档，可直接用于指导 AI 重新生成风格相似但内容不同的图片。

解析结果结构

解析完成后，助手将输出包含以下模块的 JSON 文档：

JSON 模块内容说明
metadata图片主题、尺寸、渲染引擎、色彩模式等基础元数据
typography_layer_specification文字内容、字体样式、视觉效果、位置坐标等排版规格
pixel_level_visual_specs角色/主体细节、背景场景、光照设置等像素级视觉描述
color_palette_hex图片的主要色彩方案（十六进制色值）
ai_generation_prompts多格式 AI 生成提示词，包含 DALL·E 3 优化版、Midjourney v6 格式和中文描述

图片解析操作步骤

切换到"图片解析"标签

点击面板底部的图片解析标签。

上传待解析图片

点击上传区域，选择您希望解析的参考图片（支持 JPG / PNG / WEBP）。

点击"开始解析"

图片上传后，点击开始解析按钮，助手将自动分析图片并返回 JSON 文档。

获取并使用 JSON 结果

助手将返回结构化的 JSON 文档。您可以：

  • 直接复制 ai_generation_prompts 中的提示词用于图片生成
  • 切换到对话助手标签，通过对话进一步修改 JSON 中的特定字段（如更换配色、调整文字内容）
  • 将修改后的提示词用于批量生成，探索不同的设计变体

图片解析功能特别适合借鉴参考图设计风格的场景——解析后的 JSON 提示词能够还原设计语言，但生成的新图片在内容上与原图完全不同，规避版权风险。
