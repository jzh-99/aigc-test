# Toby.AI 企业版 · AIGC 创作平台

欢迎使用 **Toby.AI 企业版 AIGC 创作平台**。本平台面向企业级内容创作场景，集成了业界领先的图像生成、视频生成与 AI 对话能力，帮助您的团队以极低的成本完成高质量的视觉内容生产。

## 核心能力

| 功能模块 | 说明 |
|---|---|
| **图片生成** | 支持文生图、图生图，提供全能图片2（5积分/张）和全能图片Pro（10积分/张）两种模型，支持批量生成 |
| **视频生成** | 支持首尾帧控制和参考图生视频两种模式，与图片生成在同一页面通过标签切换 |
| **资产库** | 统一管理所有生成的图片与视频资产，支持按日期筛选、下载、复用和删除 |
| **AI 助手** | 常驻悬浮按钮，提供对话式创作辅助和图片一键解析为结构化提示词 |

## 计费方式

平台采用**积分制**计费。所有生成功能直接消耗账户积分，无需配置 API Key 或任何第三方服务，开箱即用。

## 本手册适用对象

本手册面向**标准版 Editor 用户**，介绍创作平台的全部可用功能及典型使用场景。
