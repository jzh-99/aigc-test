# 使用样例：海报制作

本章通过一个完整的实战案例，演示如何综合运用平台的**图片生成**、**AI 助手**功能，从参考图出发，制作一张风格相似但内容全新的定制化海报。

{/* TODO: 截图 - 最终完成的海报效果图（左：参考图，右：生成结果），作为章节引导图 */}
![海报制作案例展示](/images/case-poster-result.png)

---

## 工作流程概览

```
参考海报
   ↓
[可选] 图生图：调整尺寸 / 提升清晰度
   ↓
AI 助手：图片解析 → JSON 提示词
   ↓
对话改写 JSON（替换品牌信息、调整配色等）
   ↓
图片生成：粘贴提示词 + 上传参考图 + 批量生成
   ↓
选优下载
```

---

## 详细步骤

<Steps>

### 第一步：准备参考海报

找到一张设计风格符合您需求的参考海报。

{/* TODO: 截图 - 参考海报示例图 */}
![参考海报](/images/case-poster-reference.png)

**可选操作：图生图预处理**

如果参考图尺寸比例不符合目标输出（如需要 9:16 竖版，参考图是 16:9 横版），建议先通过**图片生成**功能进行处理：

在创作生成页面，上传参考图作为参考图片，在提示词中描述目标尺寸和风格，选择合适的比例（如 9:16），提交生成。

{/* TODO: 截图 - 图片生成界面，展示参考图上传和比例设置 */}
![图生图预处理](/images/case-poster-img2img.png)

---

### 第二步：AI 助手解析参考图

点击页面右下角的 **AI 助手**悬浮按钮，打开助手面板，切换到**图片解析**标签，上传参考海报，点击**开始解析**：

{/* TODO: 截图 - AI 助手图片解析标签，已上传参考图并收到 JSON 解析结果 */}
![AI 解析海报](/images/case-poster-parse.png)

助手将返回包含配色、排版、视觉风格和多格式提示词的 JSON 文档。

---

### 第三步：通过对话改写 JSON

切换到**对话助手**标签，在同一对话窗口中，告诉助手您需要修改的内容，例如：

> 请将 JSON 中的品牌名称替换为"Toby.AI"，主色调改为深蓝色系（#0A1628），文案改为："用 AI，重新定义创作"，保持整体排版风格不变。

{/* TODO: 截图 - 对话改写 JSON 的过程，展示修改指令和助手返回的新 JSON */}
![改写 JSON](/images/case-poster-rewrite.png)

<Callout type="info">
您可以多轮对话反复调整，直到 JSON 中的各项参数符合您的预期为止。
</Callout>

---

### 第四步：生成海报

复制改写后 JSON 中 `ai_generation_prompts` 下的提示词，进入**创作生成**页面（图片模式）：

1. 选择合适的模型（推荐**全能图片Pro**，细节更丰富）
2. 将提示词粘贴到输入框
3. 如有必要，上传参考图以保留构图风格
4. 设置目标比例（如 9:16）
5. 将**数量**设为 4，开启批量生成

{/* TODO: 截图 - 图片生成页面，填好提示词和参数，即将提交的状态 */}
![图片生成配置](/images/case-poster-generate.png)

提交后，在历史记录中查看 4 张生成结果，选取最满意的一张下载。

{/* TODO: 截图 - 生成完成后的 4 张结果对比图 */}
![批量生成结果](/images/case-poster-batch-result.png)

---

### 第五步：下载与使用

点击选中的图片，在侧边栏点击**下载**，保存高清原图。

如需在此基础上继续微调，可点击**复用**，将该图片的参数填回创作面板再次调整。

</Steps>

---

## 小结

通过以上流程，您实现了：
- **借鉴**参考图的设计语言和视觉风格
- **替换**为自己的品牌信息和创意内容
- 生成的最终图片在版权上与参考图**完全独立**
