# 工作台

登录成功后，您将进入**工作台**主界面。工作台展示您的积分概况、近期生成记录和快速创作入口。

## 侧边栏导航

页面左侧为固定侧边栏，包含平台所有功能入口：

| 菜单项 | 说明 |
|---|---|
| **工作台** | 当前页面，概览统计与快速入口 |
| **创作生成** | 图片生成与视频生成（同一页面，通过标签切换） |
| **资产库** | 查看和管理所有已生成的图片与视频 |
| **团队管理** | 管理团队成员和工作区（仅团队 Owner 可见） |
| **管理后台** | 平台管理功能（仅系统管理员可见） |
| **设置** | 修改个人信息和密码 |

侧边栏底部显示当前**积分余额**，点击左下角按钮可折叠/展开侧边栏。

---

## 工作区切换（Workspace）

侧边栏顶部为**工作区切换器**。如果您被分配到多个工作区，可在此切换。

{/* TODO: 截图 - 侧边栏工作区切换器展开状态 */}
![工作区切换](/images/workspace-switch.png)

不同工作区的历史记录和资产库相互独立，切换后界面内容随之更新。

<Callout type="warning">
如果您尚未被分配到任何工作区，创作生成页面将显示提示，请联系团队负责人为您分配工作区。
</Callout>

---

## 统计卡片

工作台顶部展示三张统计卡片：

| 卡片 | 说明 |
|---|---|
| **可用积分** | 您当前可用的积分余额（Editor 用户显示个人配额剩余；Owner/管理员显示团队总余额） |
| **生成次数** | 当前工作区累计成功生成的图片/视频数量 |
| **成功率** | 生成任务的成功比例 |

---

## 最近生成

工作台中部展示最近的生成批次记录，点击任意批次可查看详情。

---

## 快速入口

工作台右侧提供快速跳转按钮，点击即可直接进入对应创作页面：

- **图片生成** → 跳转至创作生成页（图片模式）
- **视频生成** → 跳转至创作生成页（视频模式）
