# 资产库

资产库（侧边栏 → 资产库）汇聚了当前工作区内所有已生成的图片和视频，支持浏览、筛选、下载、复用和删除。

## 资产分类

资产库顶部提供**图片 / 视频**切换按钮（视频功能开启时显示），分别展示对应类型的资产。

<Callout type="info">
资产库与工作区绑定，切换工作区后显示的资产随之切换。
</Callout>

---

## 按日期筛选

顶部右侧提供**日期筛选器**，选择日期后仅显示该日期生成的资产。点击筛选器右侧的 × 按钮可清除筛选。

---

## 浏览资产

资产以**网格缩略图**形式展示，按生成日期分组，最新的在最前。

- 图片：显示缩略图
- 视频：显示视频封面帧，并叠加播放图标

页面支持**无限滚动**，滚动到底部自动加载更多资产。

## 资产操作

**鼠标悬停**在任意资产上，显示操作按钮：

| 操作 | 说明 |
|---|---|
| **复用** | 获取该资产的生成参数，跳转至创作生成页并自动填入提示词和参数 |
| **下载** | 将图片或视频保存至本地 |
| **删除** | 从资产库中删除该资产（不可恢复） |

**点击**资产可进行全屏预览：

- **图片**：打开灯箱全屏查看，支持左右切换浏览同日期资产，底部显示提示词、模型和生成时间
- **视频**：打开播放对话框，支持在线播放，底部同样显示提示词和模型信息
